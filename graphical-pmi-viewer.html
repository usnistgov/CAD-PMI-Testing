Graphical PMI Viewer

  • This is an example of visualizing of Graphical PMI. The visualization was generated by the NIST STEP File Analyzer and rendered with X3DOM.
  • The graphical PMI is from a STEP AP242 file generated from one of the NIST CAD models. Only the graphical PMI is shown. Part geometry can be viewed with STEP file viewers.
  • Use the mouse to rotate, pan, and zoom. Use Page Down to switch between perspective and orthographic views. Saved Views are ignored.
