MBE PMI Validation and Conformance Testing Project

PMI Verification Testing Results Browser

Before using the testing results browser below you should be familiar with the MBE PMI Validation and Conformance Testing Project and the verification testing reports.
  • Measuring the PMI Modeling Capability in CAD Systems: Report 1 - Combined Test Case Verification
  • Measuring the PMI Modeling Capability in CAD Systems: Report 3 - Fully-Toleranced Test Case Verification
The browser provides supplemental information for the high-level verification testing results documented in the reports and is not intended to be a first source of information for the results.

Browser

The browser below uses the open-source version of Keshif, a web-based data exploration environment for data analytics. There are six frames in the Keshif environment: CAD System, Test Case, Annotation Type, PMI Error, Error Type, and PMI Annotation.
  1. Test Case, Annotation Type, and PMI Annotation are defined in the test case browser.

How to use Keshif data exploration environment

Mousing over a text label (e.g. CAD B (2015), Position, C1) highlights the bar in orange and associated data in the other frames. This shows the relationships between the data. Clicking on the text label filters information in all frames by the selected value. Clicking on a selected value deselects it. After one text label has been selected, more can be selected by clicking 'Or' to the left of a label. Data can be filtered out by clicking 'Not' to the left of a label. PMI Annotations can also be filtered with a Search string.
